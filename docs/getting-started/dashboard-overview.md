# Dashboard Overview

Get familiar with the DulyPlan dashboard and its main features.

## Main Navigation

The dashboard includes the following sections:

- **Home** - Overview and quick stats
- **Calendar** - View and manage scheduled posts
- **Create** - Create new posts
- **Drafts** - Manage draft posts
- **Analytics** - View performance metrics
- **Inbox** - Manage messages and notifications
- **Error Logs** - Monitor system errors

## Top Navigation Bar

- **+ Create Post** - Quick access to post creation
- **Search** - Search functionality
- **Notifications** - View notifications
- **User Profile** - Access profile and settings

## Dashboard Cards

The home dashboard displays:

- **Scheduled Posts** - Upcoming scheduled content
- **Social Accounts** - Connected social media accounts
- **Draft Posts** - Your draft content
- **Messages** - Recent messages and notifications
