# Connecting Social Media Accounts

Learn how to connect your social media accounts to DulyPlan.

## Supported Platforms

DulyPlan supports the following social media platforms:

- Facebook
- Instagram
- Twitter/X
- LinkedIn
- TikTok
- Threads
- YouTube
- Pinterest
- Telegram
- WhatsApp

## How to Connect

1. Navigate to **Dashboard** â†’ **Social Accounts**
2. Click **"+ Add Account"**
3. Select the platform you want to connect
4. Follow the OAuth authorization flow
5. Grant necessary permissions
6. Your account will be connected

## Managing Accounts

- **View All Accounts** - See all connected accounts
- **Edit Account** - Update account settings
- **Disconnect Account** - Remove account connection

## Permissions Required

Each platform requires specific permissions:
- Post creation and publishing
- Media upload
- Profile information access
