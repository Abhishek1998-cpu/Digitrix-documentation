# Getting Started with DulyPlan

Welcome to DulyPlan! This guide will help you get started with the platform.

## What You'll Learn

In this section, you'll learn:

- How to create an account
- How to navigate the dashboard
- How to connect your social media accounts
- How to create and schedule your first post

## Prerequisites

Before you begin, make sure you have:

- A valid email address
- Access to your social media accounts (Facebook, Instagram, Twitter/X, LinkedIn, etc.)
- A modern web browser (Chrome, Firefox, Safari, or Edge)
