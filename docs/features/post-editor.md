# Post Editor Features

Advanced features available in the DulyPlan post editor.

## Editor Features

### Text Editing

- Rich text formatting
- Hashtag suggestions
- Mention users
- Link previews

### Media Management

- Image upload and editing
- Video upload
- Canva template integration
- Media library

### Platform-Specific Features

- Platform-specific formatting
- Character count per platform
- Platform preview
- Optimal image sizes
