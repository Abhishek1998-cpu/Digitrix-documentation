# Creating Posts

Learn how to create and publish posts on DulyPlan.

## Post Creation Overview

DulyPlan allows you to create posts for multiple social media platforms from a single interface.

## Creating a New Post

1. Navigate to **Create** in the main menu
2. Select the platforms you want to post to
3. Choose the accounts for each platform
4. Add your content (text, images, videos)
5. Preview your post
6. Schedule or publish immediately

## Post Content

### Text Content

- Write your post caption
- Add hashtags
- Mention users (platform-specific)
- Format text (bold, italic, etc.)

### Media

- Upload images
- Upload videos
- Use Canva templates
- Edit media with built-in editor

### Scheduling

- Schedule for later
- Set specific date and time
- Choose timezone
- Set up recurring posts

## Post Types

- **Text Posts** - Text-only content
- **Image Posts** - Single or multiple images
- **Video Posts** - Video content
- **Link Posts** - Posts with external links
- **Story Posts** - Platform-specific story format

## Post Status

- **Draft** - Saved but not published
- **Scheduled** - Set to publish at a future time
- **Published** - Already published
- **Failed** - Publishing error occurred
