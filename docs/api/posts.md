# Post Management API

API endpoints for creating, managing, and scheduling posts.

## Endpoints

### Create Post

```http
POST /v1/dashboard/channel/create-post
```

### Get Posts

```http
GET /v1/dashboard/channel/posts
```

### Update Post

```http
PUT /v1/dashboard/channel/posts/:postId
```

### Delete Post

```http
DELETE /v1/dashboard/channel/posts/:postId
```
