# Authentication API

API endpoints for user authentication and account management.

## Endpoints

### Register User

```http
POST /v1/auth/register
```

### Login

```http
POST /v1/auth/login
```

### Logout

```http
POST /v1/auth/logout
```

### Verify Email

```http
POST /v1/auth/verify-email
```

### Get User Details

```http
GET /v1/auth/user-details
```
